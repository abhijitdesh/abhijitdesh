Regression Analysis on MNIST Dataset

library(ggplot2)

Objective

Objective of this project is to implement logistic regression for classification of digits on MNIST image dataset and evaluate how the behavior of the model is affected when parameters and sizes of the training set is changed.

MNIST Dataset

The MNIST database (Modified National Institute of Standards and Technology database) is a large database of handwritten digits that is commonly used for training various image processing systems. Original MNIST database contains 60,000 training images and 10,000 testing images.

For this project I have chosen a subset of the original MNIST dataset, where the images of only four digits have been included (0, 1, 3 and 5). As a further simplification, I will only attempt binary classification in this project.

Dataset contains two CSV files viz. mnist_test.csv and mnist_train.csv. Each CSV file contains <785 rows x n columns>, where n is the number of sample images in the file. Each column represents one 28×28 image of a digit stacked into a 784 length vector followed by the class label (0, 1, 3 or 5).

Implementation Steps

Implementation of this project is carried out in following steps: 1. Data Preprocessing 2. Building Model 3. Predicting Labels 4. Calculating Accuracy 5. Visualizing Results

1. Data Preprocessing

In this step training and testing data is read and loaded into separate matrices. In order to seperate feature columns from label column both of the matrices are transposed. Training and Test dataset is then sudivided into two data sets. One dataset consists of labels 0 or 1 while other consists of labels 3 or 5. In both dataset label column is updated to either 1 or -1 with fllowing logic: 0 changed to -1, 1 remained as 1, 3 changed to -1 and 5 changed to 1. A new labels vector is created and feature matrix is updated to include additional bias term (set to 1). Bias term allows the model to have more flexibility.

2. Building Model

model() is a driver function which calls train(), predict() and accuracy functions on trasining and testing dataset. This functions accepts input of train_data, train_labels, test_data, test_labels, alpha. Aplha is a hyperparameter. Output of this function is a learned model parameters (theta) abd predicition accuraccy (train_acc and test_acc).

model <- function(train_data, train_labels, test_data, test_labels, alpha) { 
  theta <- train(train_data, train_labels, alpha) 
  train_predicted_labels <- predict(theta, train_data) 
  test_predicted_labels <- predict(theta, test_data) 
  train_acc <- accuracy(train_labels, train_predicted_labels) 
  test_acc <- accuracy(test_labels, test_predicted_labels) 
  result_list<- list(theta, train_acc, test_acc) 
  return(result_list) 
}

train() function determines the optimal value of parameter vector theta. This accepts input as trainings data each with 785 features, corresponding label and learning rate alpha. Output of this fucntion is parameter vector “theta” which contains the optimal weights for each of the features and bias term.

train() implements following loss function to calculate loss occurs for a particular theta vector. \[ L(\theta) = log({1 + \exp(-y^{(i)} <\theta, x^{(i)}> )}) \] where \(x^{(i)}\) is the data point represented in a vector of features, \(y^{(i)}\) is the class label, and \(\theta\) is the parameter vector.

Stochastic gradient descent algorithm is used to determine the minimum. When the loss function is at a minimum its derivative is zero. A random sample from training dataset with theta vector is used to calculate vector of partial dreviaties. These partial derivatives along with hyperparameters is used to update the vector theta.

The formula for the gradient descent: \[ \theta_j \leftarrow \theta_j - \alpha \sum_{i=1}^{n} \frac{1}{1 + \exp(-y^{(i)} <\theta, x^{(i)}> )} \] where \(x^{(i)}\) is the data point represented in a vector of features, \(y^{(i)}\) is the class label, and \(\theta\) is the parameter vector. The goal is to reach the \(\theta\) that maximizes our likelihood function (given the data we use to train the model). 

train <- function(data, labels, alpha) { 
  
  theta = runif(ncol(data)) 
  mean_epoch_loss_change = 0.0 
  min_loss_change = 0.01 
  max_num_epoch = 70 
  
  for (epoch in seq(max_num_epoch)){ 

    index_values = sample(seq(nrow(data))) 
    for (i in index_values) { 
      x <- data[i,] 
      y <- labels[i] 
      part_derv = calc_part_derv(x,y, theta) 
      theta = theta - (alpha * part_derv) 
    } 
    new_mean_epoch_loss <- calc_loss(data, labels, theta) 
    
    if (abs(mean_epoch_loss_change - new_mean_epoch_loss) < min_loss_change) { 
      break 
    } 
    mean_epoch_loss_change <- new_mean_epoch_loss 
  } 
  return(theta) 
  
}

calc_part_derv <- function(x,y, theta){ 
  part_derv = (y%*%x ) / (exp(-y*(as.numeric(theta%*%x))) +1) 
  return(part_derv) 
}

calc_loss <- function(data, labels, theta) { 
  batch_loss = 0 
  losses <- log(1+exp(labels*(as.numeric(data%*%t(theta))))) 
  batch_loss <- sum(losses) 
  return(batch_loss/nrow(data)) 
}

3. Predicting Labels

Function predict() function is used to predict labels for train data as well as test dataset.
This function accepts input as theta vector and dataset for prediction

A sigmoid function is used to predict the probability of a particular data row If a value in the resulting vector “predictions” is greater than 0.5 the label will be -1, otherwise it will 1. This function returns the vector of predicted labels

predict <- function(theta, data) { 
  predictions <- 1/(1+exp(-(data%*%t(theta)))) 
  predictions <-ifelse(predictions> 0.5, -1, 1) 
  return(predictions) 
}

4. Calculating Accuracy

This function compares two vectors, the true labels and the predicted labels and calculates prediction accurracy between 0 and 1.

accuracy <- function(labels, labels_pred) { 

  match <- labels == labels_pred 
  acc <- (sum(match) / length(labels)) 
  return(acc) 
  
}

Trainig and testing accuracy is calculated by training model with 10 various values of alpha with 10 iterations each. Selected values of alpha are 0.0001, 0.001, 0.005, 0.01, 0.05, 0.1, 0.2, 0.25, 0.3 and 0.35

alphas = c(0.0001, 0.001, 0.005, 0.01, 0.05, 0.1, 0.2, 0.25, 0.3, 0.35) 
iterations <- 10 


calc_alpha_values <- function(training_data, training_labels, testing_data, testing_labels, alphas) { 
  
  result_matrix <- matrix(nrow=(length(alphas)*iterations), ncol=3, data = NA) 
  for (a in seq(length(alphas))) { 
    for (n in seq(iterations)) { 
      result = model(training_data, training_labels, testing_data, testing_labels, alphas[a] ) 
      index = ((a-1)* iterations) + n 
      result_matrix[index,] <- c(alphas[a], result[[2]],result[[3]]) 
    } 
  } 
  
  result_df <-as.data.frame(result_matrix) 
  names(result_df) <- c("alpha", "train_acc", "test_acc") 
  
  means <- aggregate(result_df[, 2:3], list(result_df$alpha), mean) 
  sds <- aggregate(result_df[, 2:3], list(result_df$alpha), sd) 
  
  stats <- as.data.frame(cbind(means[,1], means[,2], means[,3], sds[,2], sds[,3])) 
  names(stats)  <- c("alpha", "mean_train_acc", "mean_test_acc", "sd_train_acc", "sd_test_acc") 
  
  return(stats)
  
} 


alpha_plots <- function(stats, dataset_name) { 
  
  offset = .0007

    geom_point(aes(x=log(alpha) + log_offset, y=mean_test_acc, col="Test")) +

    ylab("Accuracy") + 
    ggtitle("Train and Test accuracy for various log(alpha) values") + 
    labs(subtitle =paste("using the", dataset_name)) 
  
  print(plot_linear)

calc_learning_curves <- function(samples, iterations, training_data, training_labels, testing_data, testing_labels, alpha) { 
  
  result_matrix <- matrix(nrow=(length(samples)*iterations), ncol=3, data = NA) 
  
  for (s in seq(length(samples))) { 
    
    for (n in seq(iterations)){ 
      sample_indices <- sample(seq(nrow(training_data)), floor(nrow(training_data)*samples[s])) 
      
      sample_training_data <- training_data[sample_indices,] 
      sample_training_labels <- training_labels[sample_indices] 
      
      result = model(sample_training_data, sample_training_labels, testing_data, testing_labels, alpha ) 
      
      index = ((s-1)* iterations) + n 
      result_matrix[index,] <- c(samples[s], result[[2]],result[[3]]) 
      
    } 
  } 
  
  result_df <-as.data.frame(result_matrix) 
  names(result_df)  <- c("sample_size", "train_acc", "test_acc") 
  
  means <- aggregate(result_df[, 2:3], list(result_df$sample_size), mean) 
  sds <- aggregate(result_df[, 2:3], list(result_df$sample_size), sd) 
  
  stats <- as.data.frame(cbind(means[,1], means[,2], means[,3], sds[,2], sds[,3])) 
  names(stats)  <- c("sample_size", "mean_train_acc", "mean_test_acc", "sd_train_acc", "sd_test_acc") 
  
  return(stats) 
  
}
