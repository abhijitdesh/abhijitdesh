Machine Learning for Stock Trading¶

Part IV - Machine Learning Strategy¶

Objective: Goal of this project is to create a strategy using machine learning agent to trade a given stocks to maximize the returns. During first phase learner is feed with stock symbol and time period to learn a strategy. In testing phase learner implements the strategy to predict the stock prices for the given period. For this project a Random Forest classification algorithm with leaf size of 5 and bag size of 20 is used to implement the learning agent.

Framing the trading problem as machine learning problem

In order to maximize the returns on a stock, we must determine stock position (long or short) and pick a right time to enter and exit these positions based on the analysis of the technical indicator. To frame this problem of predicting action to optimize stock portfolio, an ensemble of Random Tree learners is used. Learner is first trained on the in-sample data and then a new out of sample data is provided to predict the action to maximize the portfolio value. Learner uses three indicators, Price to SMA ratio, Bollinger Bands and Momentum to make a decision on stock position and entry and exit criteria.

  • Simple Moving Average: Simple moving average (SMA) smooths out past price data. It calculates the rolling mean price of a stock for specified lookback periods. If stock prices move significantly from SMA, price tends to come back to SMA. Strategy is to buy stock when stock price falls significantly below the SMA and to short the stock when stock price rise significantly above SMA. We calculate the ratio of price to SMA and buy a stock when price to SMA ratio is below threshold and sell a stock when price to SMA ratio is above threshold.
  • Bollinger Bands: Bollinger Bands detects the volatility and dynamics of the price movement in the stock price. Using Bollinger Bands, we can predict whether current trend in the stock price will continue or turn around. If the price is close to or outside of the upper band, the stock price would be considered high and we should sell the stock. On the other hand, if the price is near the lower band, stock price would be considered low and we should buy the stock. Middle Band = SMA with look back window of 20 days Upper Band = SMA + (2 x standard deviation of price) Lower Band = SMA - (2 x standard deviation of price) %B = (Current Price - Lower Band) / (Upper Band - Lower Band)
  • Momentum: This technical indicator help us to predict strength of a price movement. Momentum is calculated by comparing the most recent closing price to a previous closing price over the lookback window. M = CP – CPn Where M is momentum, CP is closing price and n is lookback window. When the Momentum crosses above the zero-line stock price is reversing and price is bottomed out hence it is a buy signal. When the Momentum crosses below the zero-line stock price is reversing and price is topped out hence it is a sell signal.

Training Phase: To train the learner stock prices indexed by trading days are fed and learner creates the data frame (trainX) with 3 indicators as independent variables. Learner also creates a data frame (trainY) to represent 3 classes of dependent variables. Three classes 1, -1 and 0 in trainY represents LONG, SHORT and DO NOTHING positions.
Testing Phase: In this phase learner is giving out of sample prices data for a stock and using the three technical indicators, learner predicts the action (1, -1 or 0) which will maximize the value of portfolio.
Discretization: Random Tree learner classify the independent variables into various class of dependent variables and hence we do not need to discretize the data fed to learning algorithm. Random Tree learner built in previous project was can be reused in this project by changing median to mode to accommodate classification of input data. No discretization (standardization) is required for this implementation.

Comparision of Manual Strategy vs Machine Learning Strategy

In this experiment we compare performance of benchmark, manual strategy and machine learning strategy. Both manual strategy and machine learning strategy using the same 3 indicators mentioned in the Part 1. This experiment is carried out for following data –

  • Stock: JPM
  • In sample period: January 1, 2008 to December 31 2009.
  • Out of sample period: January 1, 2010 to December 31 2011.
  • Starting cash: 100,000.
  • Allowable positions: 1000 shares long, 1000 shares short, 0 shares.
  • Commission: 0.00
  • Impact: 0.0
    Benchmark: The performance of a portfolio starting with 100,000 cash, investing in 1000 shares of the symbol in use and holding that position. Include transaction costs. Manual Strategy: A rule based manual strategy generate trades data frame using market simulator code developed in previous project. Following logic was developed to generate buy, sell or do nothing signals. Buy or sell is triggered when all three indicators meets the thresholds.
  • Buy stock if momentum is less than 0.05 and Bollinger’s Band percentage is less than 0.15 and price to SMA ratio is less than 0.85.
  • Sell the stock if momentum is greater than 0.05 and Bollinger’s Band percentage is greater than 0.15 and price to SMA ratio is greater than 1.02.
    Machine Learning Strategy: Implements the Random Forest based classifier with bag size of 10 and leaf size of 5. Classifier uses 5 day return window to take buy, sell or do-nothing decision-based buy and sell threshold of > 0.02 and < 0.02 respectively.

Comprision of Learning Agent's Perfromance for various impact values

Hypothesis: Since impact is the cost for every buy/sell transaction, it inversely proportional to the portfolio value. It can be safely assumed that lower the impact higher the performance of portfolio.

In this experiment we run the strategy learner with changing the value of impact and analyzed the effect on the in sample trading behavior and learner’s performance. We compare Cumulative Return, Standard Deviation and Average Daily Return for multiple values of impact. This experiment is carried out for following data –

  • In sample period: January 1, 2008 to December 31 2009.
  • Out of sample period: January 1, 2010 to December 31 2011.
  • Starting cash: 100,000.
  • Allowable positions: 1000 shares long, 1000 shares short, 0 shares.
  • Commission: 0.00
  • Impact: 0.0, 0.005, 0.01, 0.015, 0.02 and 0.025
